# Victor Portfolio

Portfólio pessoal de Victor Hugo - Desenvolvedor Frontend

## 🚀 Como usar

### Desenvolvimento local
```bash
npm install
npm run dev
```

### Deploy automático (GitHub Actions)
1. Faça push para a branch `main`
2. O GitHub Actions builda e publica automaticamente
3. Acesse: `https://VictorMuza.github.io/victor-portfolio/`

## ⚙️ Configurar GitHub Pages

1. Vá em **Settings** → **Pages** no seu repositório
2. Em **Build and deployment**, selecione **GitHub Actions**
3. Salve

## 📝 Personalizar

Edite `src/data/projects.js` para atualizar seus projetos.
